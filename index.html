CitiSync – Web-Based Transaction Processing System for Palmera 3 and 4 Homeowners’ Association

CitiSync is a web-based platform designed to assist homeowners’ association officers in managing resident records and automating key processes such as monthly dues collection, document requests, and amenity reservations. The system improves administrative efficiency and provides transparency for both officers and residents.

Developed a full-stack solution with a responsive user interface and role-based access control, providing separate dashboards and permissions for Admin and Resident users. The system supports secure login, real-time updates, and mobile-friendly access for better user experience.

Technologies Used: ASP.NET Core MVC, C#, HTML5, CSS3, JavaScript, Bootstrap, jQuery, AJAX, and MySQL.
